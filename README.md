# PMD Arquitectura Web

Landing page completa con cotizadores para PMD Arquitectura, GBA Norte.

## Archivos

- `index.html` — Web principal con cotizador doble modo integrado
- `cotizador-tecnico.html` — Cotizador avanzado por rubro (cubierta, pisos, cocina, aberturas, sanitaria, electrica, calefaccion)
- `cotizador-simple.html` — Cotizador simple 3 pasos standalone

## Deploy con GitHub Pages

1. Settings → Pages → Source: main / root
2. URL: `https://augusto-pmd.github.io/pmd-arquitectura-web`

## Stack

HTML + CSS + JS puro. Sin dependencias. Funciona standalone o en Vercel/Netlify/GitHub Pages.
